# AI中医诊断系统演示案例库

本文档整理了系统中的预制病例输入描述，按疾病类型分类，方便演示者快速选择效果最佳的案例进行演示。

## 使用说明

1. **选择合适案例**：根据演示需要选择不同疾病类型的案例
2. **复制输入描述**：直接复制"患者描述"部分到系统输入框
3. **预期效果**：每个案例都有对应的预制诊断报告，展示效果良好
4. **演示建议**：标注了推荐程度，★★★为最佳演示效果
